# 2048

📄 This README is available in: [**English 🇬🇧**](README.md)

Вебзастосунок, що є клоном класичної гри **2048**, створений з використанням **React** та **TypeScript**.

🔗 **Посилання на застосунок:** https://harynets.github.io/2048/

---

## 🛠️ Використані технології

-   **React**
-   **TypeScript**
-   **Vite**
-   **CSS**
-   **HTML**
-   **Tailwind CSS**

## ✨ Функціонал

-   🎯 Класична ігрова механіка **2048**
-   📱 Підтримка свайпів на мобільних пристроях
-   🎞️ Плавні анімації руху, об’єднання та появи плиток
-   📊 Підрахунок очок та анімація оновлення рахунку
-   📐 Адаптивна верстка
-   🎨 Стилізація за допомогою **Tailwind CSS**
-   🚀 Оптимізація продуктивності з використанням **React.memo**
-   🔒 Типобезпечна логіка на **TypeScript**

## ⚙️ Встановлення та запуск

```bash
# Клонувати репозиторій
git clone https://github.com/Harynets/2048.git
cd 2048

# Встановити залежності
npm install

# Запустити сервер розробки
npm run dev
```
